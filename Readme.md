# JavaScript ES6 Abstaract Data Types Classes

## Installation

```bash
npm install antonKalinin/adt-classes --save
```

## Usage

```js
var ADT = require('adt-classes'),
    Graph = ADT.Graph.Graph;

var graph = new Graph();
```

## Data Types

  * Graph
  * BinaryHeap
  * Matrix
  * Queue
